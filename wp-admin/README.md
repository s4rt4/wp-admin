# Replika UI Dashboard Admin WordPress (Versi Bahasa Indonesia)

Proyek ini adalah replika statis dari UI (User Interface) dashboard admin WordPress/ClassicPress. Dibuat murni untuk tujuan pembelajaran front-end, menggunakan HTML, CSS, dan JavaScript.


---

## Fitur Utama

Proyek ini berhasil meniru beberapa fitur inti dari UI aslinya:

* **Sidebar Dinamis:** Sidebar dapat diciutkan (`collapse`) dan diperluas (`expand`).
* **Menu Accordion:** Saat sidebar terbuka, submenu terbuka ke bawah dengan *klik* (bukan *hover*).
* **Menu Pop-up:** Saat sidebar diciutkan, submenu muncul sebagai *pop-up* di samping saat di-*hover*.
* **Status Aktif (Highlight):** Blok biru dan "coakan" (notch) akan berpindah secara dinamis ke item menu mana pun yang sedang aktif.
* **Top Admin Bar:** Termasuk menu dropdown *hover* di sisi kanan atas untuk profil pengguna.
* **CSS Modern:** Menggunakan variabel CSS untuk manajemen tema dan warna yang mudah.

## Cara Menggunakan

1.  Clone repositori ini.
2.  Buka file `index.html` di browser Anda.

# WordPress Admin Dashboard UI Replica (English Version)

This project is a static UI (User Interface) replica of the WordPress/ClassicPress admin dashboard. It was created purely for front-end learning purposes, using HTML, CSS, and JavaScript.


---

## Core Features

This project successfully mimics several core features of the original UI:

* **Dynamic Sidebar:** The sidebar can be collapsed and expanded.
* **Accordion Menu:** When the sidebar is expanded, submenus open downwards on *click* (not *hover*).
* **Pop-up Menu:** When the sidebar is collapsed, submenus appear as a *pop-up* on the side on *hover*.
* **Active State (Highlight):** The blue highlight block and the "notch" dynamically move to whichever menu item is currently active.
* **Top Admin Bar:** Includes the top-right hover dropdown menu for the user profile.
* **Modern CSS:** Uses CSS variables for easy theme and color management.

## How To Use

1.  Clone this repository.
2.  Open the `index.html` file in your browser.
